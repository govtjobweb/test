# TurboSpeed CDN Assets

This repository contains static assets (CSS, JS, Images, Fonts) automatically 
exported by **TurboSpeed** — a WordPress performance optimization plugin.

**Site:** https://rojgarmandi.com  

## ⚡ What TurboSpeed Does

- 🎨 **CSS & JS Optimization** — Minify, combine and defer scripts for faster load
- 🖼️ **Image Optimization** — Auto WebP conversion and lazy loading
- 🔤 **Font Optimization** — Local Google Fonts, preload and display swap
- 🌐 **GitHub CDN Export** — Push static assets to GitHub, serve via CDN
- ⚡ **Page Cache** — Full page caching for lightning fast response
- 🔒 **Security** — Brute force protection, login hardening, file editor disable
- 🧹 **Database Cleanup** — Auto remove post revisions, transients, spam comments
- 📦 **Static Export** — Export full WordPress site as static HTML
- 🔗 **DNS Prefetch & Preload** — Faster third-party resource loading

🔌 [TurboSpeed on WordPress.org](https://wordpress.org/plugins/turbospeed)
